# 需求

我打算迁移到Qdrant了，我的Qdrant安装路径在：

```
D:\qdrant-x86_64-pc-windows-msvc\qdrant.exe
```

你需要上网搜索Qdrant启动的方式和它的yaml配置文件的编写，将数据存储路径设置为当前文件夹下的`storage`文件夹。你还需要完善`start-qdrant.bat`，以下为示例，不一定对，你需要相应更改。

```
D:\qdrant-x86_64-pc-windows-msvc\qdrant.exe --config-path "D:\FastAPI-chatbox-langraph-version\backend\Qdrant-config-data\config.yaml"
```

当前文件夹下的static文件夹是`qdrant-web-ui`

示例如下，但不一定正确。

```yaml
# config.yaml
log_level: INFO

service:
  host: 0.0.0.0
  http_port: 6333      # HTTP API 端口
  grpc_port: 6334      # gRPC 端口

storage:
  storage_path: ./storage          # 数据存储路径
  snapshots_path: ./snapshots      # 快照存储路径
  on_disk_payload: true            # 关键设置，可大幅节省内存

performance:
  max_search_threads: 0            # 0 表示自动使用所有 CPU 核心

cluster:
  enabled: false                   # 单机运行必须为 false
```





# 使用方法

健康检查通过：`http://localhost:6333/healthz` 返回 200，Web UI 可通过 `http://localhost:6333/dashboard` 访问。
